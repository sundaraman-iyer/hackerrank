Logic:

63 is the number

Increment the counter at index 63 by 1

[0] * 100

final code:

def countingSort(arr):
    # Write your code here
    
    count = [0] *100
    
    for i in arr:  # 67
        count[i] +=1 # 67th index will be incremented by 1
        
    return count
